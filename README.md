# 🤖 AI 文字偵測器

這是一個簡易的 Streamlit 應用程式，旨在協助用戶偵測文本內容是否可能由 AI 生成。本工具僅供教學、示範和初步評估使用，不應作為學術違規或懲處判斷的依據。

## ✨ 功能特色

-   **文字輸入與分析**：直接貼上文本，估計其 AI 生成的可能性。
-   **可疑段落標示**：分析結果會逐句標示文本中 AI 生成機率較高的句子。
-   **美觀的進度條**：直觀地顯示整體 AI 生成可能性分數，並搭配漸層色彩變化。
-   **簡潔的使用者介面**：介面設計參考 Justdone AI Detector，簡潔易用。

## 🚀 開始使用

### 1. 安裝必要套件

請確保您的環境中已安裝 Python。然後，使用以下指令安裝所有必要的套件：

```bash
pip install -r requirements.txt
```

### 2. 執行應用程式

安裝完成後，在專案根目錄下執行以下指令即可啟動應用程式：

```bash
streamlit run app.py
```

應用程式將會在您的預設網頁瀏覽器中開啟。

## 💡 使用說明

1.  在應用程式介面中央的文字輸入框中貼上您想要分析的文本。
2.  (選填) 您可以透過滑桿調整建議的最少字元數。建議輸入至少 200 字元以獲得更可靠的偵測結果。
3.  點擊「🔍 分析文字」按鈕。
4.  應用程式將顯示整體 AI 生成可能性分數，並在下方以不同顏色標示出文本中 AI 生成機率較高的句子。

## ⚠️ 免責聲明

-   本工具使用的模型為 `openai-community/roberta-base-openai-detector`，該模型主要針對 GPT-2 類型的輸出進行訓練。
-   對於 GPT-3.5、GPT-4 或其他更新的模型，以及中文內容，偵測的準確度可能有限。
-   請勿將本工具的分析結果作為任何學術、法律或懲處判斷的唯一依據。
